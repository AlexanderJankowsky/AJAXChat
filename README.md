### __AJAXChat Standalone__
Refactored for PHP 8.2 with modern browser sound handling.<br>
Clean efficient presentation layout to suit large screen monitors.<br>
Compatible and working in both PHP 7.4 + 8.2<br>

We have used this routinely for years. It is ideal for family to keep in touch.<br>
We set the database to automatically clear messages at midnight each day.<br>
You could also set the config.php to clear messages after one or more days.<br>

Browsers store the user preferences and some layout settings in cookies.<br>
You may need to clear site cookies for some changes to carry through.<br>
